Cohort Discovery helps you quickly see whether relevant patient cohorts exist,
where data is held, and the approximate patient numbers available—without
contacting each organisation individually. Using natural language input, you can
run real-time queries across multiple pseudonymised datasets, receive simple
numerical counts, and connect directly with the appropriate data custodians to
take next steps.
