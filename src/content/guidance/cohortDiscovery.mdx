<SectionTitle title='Query Building' />
<SwimLaneContent>

Begin your query by typing a research question or by building it step-by-step using the add element buttons from your left hand tool menu. Start by adding a rule.

</SwimLaneContent>

<SwimLane >
<SectionTitle title='Component Terms' />
<SwimLaneContent>
**Rule**

Use to add a condition, measurement, medication, observation procedure or demographic to the query builder.

**Group**

Use this to combine multiple rules into a single logical unit and help structure complex queries.

**Operator**

Use to set how rules or groups relate to each other.

</SwimLaneContent>

</SwimLane>

<SwimLane>
  <SectionTitle title="Component refining" />
  <SwimLaneContent>

Edit all values and adjustments for components — including rules, groups, and
operators — in the right-hand adjustment panel.

</SwimLaneContent>
</SwimLane>

<SwimLane>
  <SectionTitle title="Understanding the Hierarchy" />
  <SwimLaneContent>
    Users can drag and drop any item in the hierarchy or visual interface to
    reorganise the structure, including moving rules and groups. Multiple items
    can be selected at once using Shift selection, allowing users to reposition
    several rules or groups together in a single drag and drop action.
  </SwimLaneContent>
</SwimLane>
