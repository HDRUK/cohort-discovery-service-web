<SectionTitle title='Query Building' />
Begin your query by typing a research question or by building it step-by-step using the add element buttons from your left hand tool menu, start by adding a rule.

</SwimLane>

<SwimLane >
<SectionTitle title='Component Terms' />

**Rule**

Use to add a condition to the query builder.

**Group**
Use to combine multiple rules into one logical unit

**Operator**
Use to set how rules or groups relate to eachother

</SwimLane>

<SwimLane >
<SectionTitle title='Component refining' />
Edit all values and adjustments for components—including rules, groups, and operators—in the right-hand adjustment panel.
</SwimLane>

<SwimLane >
<SectionTitle title='Understanding the Hierarchy' />
Users can drag and drop any item in the hierarchy or visual interface to reorganise the structure, including moving rules, groups, and sequential groups. Multiple items can be selected at once using Shift selection, allowing users to reposition several rules or groups together in a single drag-and-drop action.
</SwimLane>

<SwimLane >
<SectionTitle title='Running Queries' />
To run the query, click on the arrow next to search bar. Alternatively click on run query at the bottom of the page. 

</SwimLane>
