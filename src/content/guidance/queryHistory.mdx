Please note that results show rounded cohort counts not exact patient numbers. Low count suppression is applied to results to protect identification, so '0' counts may mean no results or be a suppressed low value. This is set by each data collection, but typically is for counts below 10.

# Statuses

**Pending** — The query is currently running and are awaiting the return of results.

**Completed** — The query has finished for this collection and the total count is finalised.

**Error** — The query could not be completed for this collection due to an issue on the data custodian side. Please re-run the query and if issues persist please feel free to contact the Gateway Helpdesk.
