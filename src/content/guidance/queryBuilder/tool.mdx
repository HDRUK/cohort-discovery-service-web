# Query Building
Begin your query by typing a research question in the Natural Language search box, or by building it step-by-step using the buttons under ‘Insert’ from the left-hand menu (e.g. starting by adding a rule).

 If the Natural Language query identifies multiple alternatives, please select your intended term from the query builder to continue, or manually refine the query using the query builder tools.
# Component Terms

## Rule

Use to add a condition, measurement, medication, observation procedure or demographic to the query builder.

## Group

Use this to combine multiple rules into a single logical unit and help structure complex queries.

## Operator

Use to set how rules or groups relate to each other.

# Component refining

Edit all values and adjustments for components — including rules, groups, and operators — in the right-hand adjustment panel.

# Understanding the Hierarchy

Users can drag and drop any item in the hierarchy or visual interface to reorganise the structure, including moving rules, groups, and sequential groups. Multiple items can be selected at once using Shift selection, allowing you to reposition several rules or groups together in a single drag and drop action.

# Running Queries

To run the query, click on the run query button next to the search bar.
