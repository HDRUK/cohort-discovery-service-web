    <SectionTitle title='Component Terms' />
    <SwimLaneContent>
      **Rule**

      Use to add a condition to the query builder. Search by term or OMOP ID.

      **Group**

      Use to combine multiple rules into one logical unit.

      **Operator**

      Use to connect rules or groups through operators like AND/OR.
    </SwimLaneContent>
  </SwimLane>
